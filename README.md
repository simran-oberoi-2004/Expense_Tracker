# 💰 Expense Tracker – Personal Finance Management System

## 📌 Overview

Expense Tracker is a **web-based personal finance management application** designed to help users efficiently track, categorize, and manage their daily expenses. The system provides a **structured dashboard, category-wise expense organization, and user authentication pages**, enabling better financial planning and visibility.

---

## ✨ Features

* 🔐 **User Authentication UI** – Signup, login, and reset password pages
* 💸 **Expense Management** – Add, track, and manage expenses
* 🗂️ **Category Management** – Organize expenses into custom categories
* 📊 **Dashboard View** – Overview of financial activity
* 📄 **Multiple UI Pages** – Dedicated pages for smooth navigation
* 📱 **Responsive Design** – Clean and user-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Architecture:** Multi-page structured UI
* *(Backend integration can be added in future)*

---

## 📂 Project Structure

* `index.html` – Landing page
* `signup.html` – User registration page
* `Reset-passwordPage.html` – Password recovery page
* `dashboard.html` – Main user dashboard
* `add-expense.html` – Add new expenses
* `add-category.html` – Create expense categories
* `all-categories.html` – View all categories
* `confirmation-page.html` – Action confirmation UI
* `blogs.html` – Informational/blog section

---

## 🚀 How It Works

1. User signs up or logs in
2. Navigates to the dashboard
3. Adds expense categories
4. Records expenses under categories
5. Views and manages financial data through the dashboard
